# AI Machine Data Analyzer

## 📌 Overview
This project simulates a machine monitoring system using Python.
It analyzes machine data such as temperature, pressure, and cycle time, and detects anomalies automatically using AI (Isolation Forest).

---

## ⚙️ Features
- Load machine data from CSV (simulating PLC data)
- Visualize machine parameters
- Detect anomalies automatically using AI
